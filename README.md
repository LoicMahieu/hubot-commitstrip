# hubot-commitstrip

Random CommitStrip

See [`src/commitstrip.coffee`](src/commitstrip.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-commitstrip --save`

Then add **hubot-commitstrip** to your `external-scripts.json`:

```json
[
  "hubot-commitstrip"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
